## ⚡ Quick Start

### Prerequisites

- Node.js 18+ 
- npm / yarn / pnpm / bun

### Installation

1. **Clone the repository:**
   ```bash
   git clone https://github.com/BitlaUmesh/Portfolio_Antigravity.git
   cd Portfolio_Antigravity
   ```

2. **Install dependencies:**
   ```bash
   npm install
   ```

3. **Configure Environment:**
   Create a `.env.local` file and add your Web3Forms key for the contact form:
   ```env
   NEXT_PUBLIC_WEB3FORMS_KEY=your_key_here
   ```

4. **Launch Development Server:**
   ```bash
   npm run dev
   ```

Open [http://localhost:3000](http://localhost:3000) to witness the experience.
